Bienvenido al Sistema de Historia Clínica del CAU - UNSAM

Esta aplicación está diseñada para uso interno de profesionales de la salud del Centro de Asistencia Universitaria (CAU) de la Universidad Nacional de San Martín.

El sistema permite registrar y consultar historias clínicas de pacientes de forma segura, utilizando técnicas de integridad basadas en blockchain. Los datos personales se almacenan en una base de datos privada, mientras que los identificadores de integridad (hashes) se vinculan con una red blockchain de capa 2.

¿Cómo utilizar la plataforma?

  • Registrar: Permite ingresar una nueva historia clínica con datos del paciente.
  • Buscar: Puede buscarse por nombre o DNI y validar la integridad de historias pasadas.

Use las opciones en la barra superior para navegar. Esta plataforma está en fase de pruebas controladas.
